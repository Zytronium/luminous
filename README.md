# Luminous
The exclusive chat app for students and graduates of [Atlas School](https://www.atlasschool.com/).
Build by alumni for alumni.

---


## Plan Overview
Using Next.js, React, Tailwind CSS, TypeScript, and a (maybe) custom SQL database,
Luminous will create a Discord server clone for students and alumni. That's right,
not a Discord clone, a Discord *server* clone. Only one server, but multiple channels
and private DMs between users.

### Stretch goals:
- Voice chat and/or video calls
- E2E encryption
- Cross-platform compatibility with Discord and maybe Root via Discord bot/webhook
- Premium paid features and/or donations to help support infrastructure costs (maybe)

### Expected Challenges:
- Designing and hosting a custom encrypted database
- Implementing stretch goals
- Affording infrastructure costs for image and video hosting

## Planned Features
- Account creation for @atlasstudents.com emails only (emails will be verified)
- Just about everything Discord has for free except servers - it will be one big server 
- DMs and Group Chats
- Custom emoji anyone can upload & send for mod approval
